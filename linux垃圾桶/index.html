rm垃圾桶

rm命令杀伤力很大，如果误删了系统文件，会很麻烦。所以，给linux系统建立一个垃圾桶，是很有必要的，详情参考Make “rm” Command To Move The Files To “Trash Can” 。

1、下载安装saferm.sh

1
2
3

git clone https://github.com/lagerspetz/linux-stuff
sudo mv linux-stuff/scripts/saferm.sh /bin
rm -Rf linux-stuff

2、编辑.bashrc文件，末尾添加

1
2

alias rm=saferm.sh
alias sudo='sudo '

3、使环境生效 source .bashrc 之后，再使用rm或者sudo rm命令，实际上执行的是mv操作，移动文件到Trash目录。
